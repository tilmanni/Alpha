Commands for Double Evaluation

Alpha:
    
    <ALPHA_JAR> -i "<PATH>/Evaluation/PUP/Double/instances/double-<NUMBER>.asp" -i "<PATH>/Evaluation/PUP/encoding-alpha.asp" -n 1 -stat -f assign
 
Qh-alpha:
 
    <ALPHA_JAR> -i "<PATH>/Evaluation/PUP/Double/instances/double-<NUMBER>.asp" -i "<PATH>/Evaluation/PUP/Double/encoding-alpha-prologheu-double.asp" -n 1 -uqh -stat -f assigned_zone_unit -f assigned_sensor_unit
 
    
Clingo:

    clingo <PATH>/Evaluation/PUP/Double/instances/double-<NUMBER>.asp <PATH>/Evaluation/PUP/encoding-hclingo.asp --quiet
    
Replace <ALPHA_JAR> with alpha jar (or use subsequent part as command line arguments).
Replace <PATH> with path to Evaluation folder.
Replace <NUMBER> with number in {20, 40, 60, 80, 100, 120, 140, 160, 180, 200},
signifying the number of comUnits in the instance.
